Изучаем C++: std::decay

Что такое std::decay?

std::decay — это стандартный метафункция в C++, которая преобразует типы, удаляя ссылки и константные квалификаторы, а также заменяя массивы и функции на указатели. Эта метафункция полезна, когда вам нужно получить "чистый" тип, который можно использовать в шаблонах и других контекстах.

Пример использования std::decay

        
        #include <iostream>
        #include <type_traits>
        
        template <typename T>
        void printType() {
            std::cout << "Тип: " << typeid(typename std::decay<T>::type).name() << std::endl;
        }
        
        int main() {
            int arr[10];
            printType<decltype(arr)>(); // Вывод: Тип: Pii (указатель на int)
        
            const int& ref = 42;
            printType<decltype(ref)>(); // Вывод: Тип: i (int)
        
            return 0;
        }

Когда использовать std::decay?

Используйте std::decay, когда вам нужно получить тип без ссылок и константных квалификаторов, а также когда вы хотите заменить массивы и функции на указатели. Это может быть полезно в шаблонах функций, где вы хотите обработать аргументы независимо от их исходного типа.
